# QPaint
Растовый редактор написанный на Python 3.6 + PyQt5.

# Функционал
  - Загрузка  / сохранение изображений
  - Задание и изменение размеров холста
  - Разные инструменты и их настройка
  - Выбор палитры

# Запуск 
> C:\QPaint>python main.py
